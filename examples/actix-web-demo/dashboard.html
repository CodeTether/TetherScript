Native Rust handler

    database::country(pool.get_ref(), &code).await
}

tetherscript handler

fn handle(request) {
    let sql = "SELECT ... WHERE code = UPPER($1)"
    let rows = db.query(sql, [request.params.code]).unwrap()
    let response = map()
    response.body = json_encode(rows[0])
    return response
}

Request and safety model

The script receives method, path, query, headers, params, and body. Its response status, headers, and body are checked before Actix sends them.

Scripts are validated at startup, execute with instruction/output budgets, and have no ambient database or network authority. Rust grants those explicitly through a PluginHost, which is where an application can inject a repository or connection pool.

The in-page benchmark performs 250 requests per route at concurrency 16 after a warm-up. For longer, alternating multi-round measurements, use the documented benchmark.js CLI.
